What it does?

Helps you bootstrap design-systems and prototypes by providing a set of non-blocking, atomic, utility classes.

That's it?

It also soft resets HTML elements and sets a basic typescale. Leaves UI untouched.

Development

  • git clone https://github.com/krisztianpuska/fractures.git
  • npm install, start, watch
  • npm build, test
  • npm run demo-build builds demo site
  • npm run demo-watch watches demo site

Download

  • download vanilla → fractures.min.css
  • download prefixed for latest 2 browser version → fractures.prefixed.min.css
